# 第九章：查找

# 基本概念

  • 查找：在数据集合中寻找满足某种条件的数据元素的过程
  • 查找表：用于查找的数据集合，由同一种数据类型（或记录）的组成，可以是一个数组或链表等数据类型
  • 查找表操作：
    • 查询某个特定的数据元素是否在查找表中（静态查找表）
    • 检索满足条件的某个特定的数据元素的各种属性（静态查找表）
    • 在查找表中插入一个数据元素（动态查找表）
    • 在查找表中删除一个数据元素（动态查找表）
  • 关键字：数据元素中唯一标识该元素的某个数据项的值，使用基于关键字的查找，查找结果应该是唯一的
