# 移动端/H5

# 王者荣耀官网

  • 说明

    这是我练习移动端网站开发的demo项目，只做了首页、新闻详情页、英雄详情页，前端页面展示内容可在后台进行配置

  • 技术栈

    1. 移动端开发（rem适配）
    2. vue.js
    3. node.js
    4. express
    5. mongodb

  • 项目组成

    1. 前端网站（web）
    2. 简易后台管理系统（admin）
    3. 服务端（server）

  • 技术点

    1. 首屏加载优化
      • webpack-bundle-analyzer分析打包大小
      • ui组件按需引用
      • 路由改成懒加载方式引入
      • cdn优化，vue全家桶和一些不常变动的库走cdn，图片因为没钱所以没走cdn，将就一下吧
      • nginx开启gzip（这个最重要！）
    2. 移动端适配
    3. 自定义小型样式库，类似bootstrap
    4. 首页轮播图，新版本vue-awesome-swiper的坑很多
    5. vue组件模块封装
    6. 富文本编辑器
    7. 遵循H5 CSS3最新规范

  • 演示：https://moba.liang2uv.top/

  • 源码：https://github.com/Liang2uv/NODE-VUE-MOBA
